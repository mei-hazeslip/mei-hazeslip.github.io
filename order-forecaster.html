Retail Forecast

Order Forecaster

Team project with Xuelian Liu, Tianzhu Qin, Wenbo Ma, and Terry Lou.

Code

Goal

Built a machine learning forecasting model to predict daily grocery delivery orders across multiple warehouses, helping reduce overstock, stockouts, and operational cost.

What I did

Analyzed historical warehouse order patterns, engineered time/holiday/seasonality features, clustered warehouses using DTW-based similarity, and compared Linear Regression, Random Forest, XGBoost, Gradient Boosting, CatBoost, and LightGBM models.

Result

The best LightGBM model reduced RMSE by 43% compared with a Linear Regression benchmark for one warehouse cluster, corresponding to an estimated $5M annual cost saving under a simplified $50-per-order business assumption.
